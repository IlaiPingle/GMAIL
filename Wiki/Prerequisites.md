# Prerequisites

Required:
- Git
- Docker + Docker Compose
- Node.js 20.x (backend + frontend build)
- npm 10+
- Java 17 + Android Studio (for Androidproject)
- CMake 3.16+ and g++ (if building Bloom service locally)
- MongoDB (only if running backend without Docker)
- WSL2 (recommended on Windows for Docker performance)

Check versions:
```
node -v
npm -v
docker --version
cmake --version
java -version
```
